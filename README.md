# UnheardEngine
 A Vulkan-based rendering engine built for advanced graphics experimentation and engine-level system design. <br>
 Implements mesh shader–driven pipelines and real-time ray tracing, with a strong focus on performance, scalability, and cross-platform rendering architecture. <br><br>
 By now it supports both Windows and Linux platform! <br><br>
 Full project download link: https://mega.nz/folder/stgwzQib#ct2EQZm5_w_M53RizRlGHw <br>
 (Only UnheardEngine.zip is needed) <br>
 If you have hard time running this project, check the SDK Dependency.txt and install necessary SDKs. <br>
